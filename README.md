# Fall Retreat 2026 – Vercel Ready

This is a Next.js project prepared for Vercel deployment.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy to Vercel

1. Push this folder to a new GitHub repository.
2. In Vercel, click **Add New Project**.
3. Import the repository.
4. Vercel will detect Next.js automatically.
5. Click **Deploy**.

## Notes

- The Roycroft font is loaded through Adobe Typekit in `app/globals.css`.
- Images are pulled from the public URLs you provided.
- Registration button links to Church Center.
